Choose a Design Approach

Three distinct approaches to improve the onboarding experience. Each design follows the project's design system with proper date pickers, visual feedback, and mobile-first responsive layouts.

Design A

Wizard Stepper

Multi-step wizard with visual progress bar. One focused section per step with clear navigation. Includes a final review step before submission.

Design B

Single Page Hero

Enhanced single-page design with visual hero header and distinct section cards. Floating circular progress indicator and weight goal visualization.

Hero Header Section Cards Floating Progress Weight Viz
Design C

Conversational Card Flow

One question per card in a conversational style. Cards fade as answered, auto-scrolling to the next. Feels like a guided conversation.
